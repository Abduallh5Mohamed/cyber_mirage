## 🚀 خطة البداية السريعة لكل الأدوار

### الأسبوع الأول:

| اليوم | الجميع | المهمة |
|-------|--------|--------|
| 1 | ✅ | تثبيت Git, Docker, Python, VS Code |
| 2 | ✅ | Clone المشروع وتشغيل `docker-compose up -d` |
| 3 | ✅ | قراءة الملفات الخاصة بالدور |
| 4 | ✅ | تجربة الأدوات الأساسية |
| 5 | ✅ | إجراء أول اختبار على النظام |

### أوامر البداية:

```bash
# 1. Clone المشروع
git clone https://github.com/Abduallh5Mohamed/cyber_mirage.git
cd cyber_mirage

# 2. تشغيل النظام
docker-compose up -d

# 3. التأكد من عمل الحاويات
docker-compose ps

# 4. فتح Dashboard
# http://localhost:8501

# 5. اختبار Honeypots
nc localhost 2222  # SSH
nc localhost 2121  # FTP
curl localhost:8080  # HTTP
```
